# Python Mobile Contacts

A simple CLI-based Contact Management System built using Python. This project allows users to store, search, and manage contacts efficiently through a menu-driven interface.

## Features
* Add new contacts
* View all contacts
* Search for a contact
* Delete a contact
* Easy-to-use CLI interface

## Technologies Used
* Python
* Dictionary (Data Structure)
* Functions
* Loops & Conditional Statements

## How to Run

1. Clone the repository:

```
git clone https://github.com/dhanushasus06-svg/Python_Mobile_Contacts.git
```

2. Navigate to the folder:

```
cd Python_Mobile_Contacts
```

3. Run the program:

```
python contact_book.py
```
